== Description ==

Vacature plugin van JobPromo

Gebruik de shortcode [jp_vacatures] om de vacature lijst op een pagina te zetten en [jp_vacature_pagina] om de vacature
beschrijving te laten zien.

== Installation ==

De administratie pagina van de Job Promo plugin is te gebruiken om de plugin aan te passen.
